## Reporting a Vulnerability

To report a vulnerability in Moveshop, please follow these steps:

1. **Do not open a public GitHub issue.** This could expose the vulnerability to others.
2. Email us at **security@moveshop.com** with the subject line: `[SECURITY] Vulnerability Report`.
3. Include a detailed description of the vulnerability, steps to reproduce it, and any potential impact.

### What to expect:
- You will receive an acknowledgment within **48 hours**.
- We aim to provide a fix or mitigation within **7–14 days**, depending on severity.
- If the vulnerability is accepted, we will credit you in the release notes (if you wish).
- If declined, we will explain why it does not meet our security criteria.

Thank you for helping keep Moveshop secure!
